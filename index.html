Current flagship project

TableArc

A local-first AI tabletop RPG engine

TableArc combines natural-language play, structured runtime state, local language-model integration, rules-aware workflows, tactical maps, combat, continuity, and debug transparency.

The project explores how an AI-assisted Dungeon Master can feel imaginative and responsive while still being grounded in explicit game state, character authority, scene structure, mechanics, and traceable decisions.

  • Living AI-assisted play loop
  • Structured campaign and character state
  • Local model support through Ollama
  • Adventure, character, rules, recap, and debug workflows
  • Map, combat, and continuity systems
