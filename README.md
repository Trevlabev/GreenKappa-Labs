# GreenKappa Labs Website

Static GitHub Pages site for GreenKappa Labs with a secondary TableArc project page.

## Structure

- `index.html` — GreenKappa Labs landing page
- `tablearc/index.html` — TableArc project page
- `style.css` — shared responsive styling
- `script.js` — footer year script
- `assets/greenkappa-logo.png` — GreenKappa Labs logo
- `assets/tablearc-banner-dark.png` — TableArc dark banner
- `assets/tablearc-banner-light.png` — TableArc light banner
- `assets/tablearc-logo-dark.png` — TableArc dark full logo
- `assets/tablearc-logo-light.png` — TableArc light full logo
- `CNAME` — custom domain setting for GitHub Pages

## GitHub Pages setup

1. Extract this ZIP into the repository root.
2. Commit and push the files to the `main` branch.
3. In GitHub, open **Settings → Pages**.
4. Set source to **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save.
