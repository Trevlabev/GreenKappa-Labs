# GreenKappa Labs Website

A static one-page website for GreenKappa Labs, designed for GitHub Pages.

## Files

- `index.html` — main page
- `style.css` — visual design
- `script.js` — small footer year script
- `assets/greenkappa-logo.png` — logo asset

## GitHub Pages setup

1. Push this folder to the `greenkappa-labs` repository.
2. Open the repository on GitHub.
3. Go to **Settings → Pages**.
4. Set source to **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save.

The site should publish at:

`https://trevlabev.github.io/greenkappa-labs/`

## Customization

Update the LinkedIn and email links in the Contact section of `index.html`.
